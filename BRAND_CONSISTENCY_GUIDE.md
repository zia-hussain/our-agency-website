## ðŸŽ¨ VISUAL BRAND GUIDELINES

### **Logo Usage**

**Primary Logo:**
```

Use: Website, proposals, presentations
```

**Variations:**
- Full color on light backgrounds
- White version on dark backgrounds
- Icon only for small spaces (favicon)

**DON'T:**
- Stretch or distort
- Change colors
- Add effects (shadows, gradients)
- Use low-resolution versions

---

### **Color Palette**

**Primary Color:**
```
Beige/Brown: #C48A64
Use: CTAs, accents, highlights, hover states
```

**Background Colors:**
```
Dark: #0A0A0A (or similar dark tone)
Card: rgba(255, 255, 255, 0.05) with backdrop blur
```

**Text Colors:**
```
Foreground: #FFFFFF (white)
Muted: #A0A0A0 (gray)
Primary: #C48A64 (brand color)
```

**Border Colors:**
```
Border: rgba(255, 255, 255, 0.1)
```

---

### **Typography**

**Headings:**
```
Weight: Bold (700)
Style: Clean, modern sans-serif
Hierarchy: Clear size progression
```

**Body Text:**
```
Weight: Regular (400)
Size: 16px base
Line Height: 1.6-1.8
```

**Code/Technical:**
```
Font: Inter
Use: Code snippets, technical details
```

---

### **Design Principles**

**Style:**
```
- Premium and sophisticated
- Clean and minimal
- Modern and tech-forward
- Professional but approachable
```

**Avoid:**
```
- Cluttered layouts
- Cheap stock photos
- Generic templates
- Aggressive colors (neon, etc.)
- Comic Sans (obviously)
```
